# Changelog

All notable changes to CostGate npm packages and Gate binaries.

Format follows [Keep a Changelog](https://keepachangelog.com/). Versions are committed in `packages/*/package.json` before each `v*` tag push.

## [1.0.0] - 2026-07-20

- First public OSS release (initial stable release).
- Publishes `@costgate/schema`, `@costgate/probe`, and `@costgate/cli` to npm.
- Publishes `costgate-gate` binaries on GitHub Releases.

## Pre-release note

- Versions used before `1.0.0` were internal development versions and were not treated as public OSS releases.
