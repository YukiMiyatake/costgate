# Documentation languages

CostGate documentation is available in **English** (default) and **Japanese**.

| Language | Location | Notes |
|----------|----------|-------|
| **English** | Repository root + `docs/` | Primary; linked from GitHub README |
| **日本語** | [`docs/ja/`](./ja/README.md) + [`README.ja.md`](../README.ja.md) | Mirrors user-facing docs |

## Conventions

- **English** files live at the canonical path (e.g. `docs/dashboard.md`).
- **Japanese** files live under `docs/ja/` with the same relative path.
- Each paired doc should link to its translation at the top.
- Roadmap, architecture, benchmarks, and log schema remain English-first; Japanese summaries may be added later.
