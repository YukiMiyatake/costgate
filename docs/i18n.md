| **日本語** | [`docs/ja/`](./ja/README.md) + [`README.ja.md`](../README.ja.md) | Mirrors for all user-facing and developer docs |

## Conventions

- **English** files live at the canonical path (e.g. `docs/dashboard.md`).
- **Japanese** files live under `docs/ja/` with the same relative path.
- Each paired doc links to its translation at the top.
- Some canonical paths (e.g. `benchmarks.md`, several `dev/*.md`) are written in Japanese; the English line in the header points to the `docs/ja/` mirror as the primary Japanese entry.

## English-only (intentional)

| Document | Reason |
|----------|--------|
| [CHANGELOG.md](../CHANGELOG.md) | Release notes; English standard |
| Package READMEs (`gate`, `probe`, `schema`) | Technical API reference; low traffic |
